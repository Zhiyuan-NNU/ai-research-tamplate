# 可选领域资料（按需使用）

本目录存放**特定研究方向可能用到的参考资料**，与通用模板分开，避免不相关方向的项目被无关文件干扰。

## 使用方式
- 如果你的研究方向与这里某个子目录相关（如电力系统会用到 MATPOWER），初始化时把对应子目录的文件**拷贝到你项目的 `代码/参考资料/` 下**备用，并在 `代码总索引.md` 里登记。
- 与你方向无关的，忽略即可，不要拷进项目。
- 判断原则：**只要有可能用到就先拷下来放好**，用得上省去临时找资料，用不上也不占项目主目录。

## 现有资料
- `电力-MATPOWER/`：MATPOWER / MPNG 的 API 详细参考手册、开发指南、代码结构分析——做电力系统、电网建模与仿真时很可能用到。

## 扩展（这个目录是开放的）
其他方向（能源材料、优化算法、机器学习、控制等）积累的可复用参考资料，也欢迎按 `方向-工具` 的命名新建子目录放进来，逐步沉淀成课题组的共享资料库。
