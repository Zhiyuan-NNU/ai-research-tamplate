# 读文献规范

> **用途**：规定本项目文献检索与精读的统一做法，保证文献工作可复现、引用可靠。
> （本文件是规范，基本固定；检索式或要求有更新时才改。精读记录写入 `文献记录表.md`。）

## 一、检索
- 中文用**知网**、英文用 **WOS**，两边都要覆盖。
- 知网专业检索字段：`SU`=主题、`TI`=篇名、`KY`=关键词、`AB`=摘要、`FT`=全文、`AU`=作者 等；用 `and / or` 组合。**知网检索式不能用 `*`**。
- WOS 用 `TS=`（主题）等字段。
- 每次检索把“检索式 + 数据库 + 日期 + 命中数”记录下来，保证检索可复现。

## 二、精读时记什么（写入 `文献记录表.md`）
- 核心贡献、方法、关键参数 / 数据、可借鉴点、与本项目的关系。
- 用**自己的话**概括，不要直接粘摘要。
- 只有真正读过正文才标“已精读”。

## 三、引用要求（防幻觉引用，重点）
- 每条引用必须能在 `文献记录表.md` 里找到、且有**真实 DOI**——**禁止编造引用**。
- 不确定文献是否真实存在时，先核实 DOI 再引用，宁缺毋造。
- 引用格式按目标期刊要求（国标 / APA / Nature 等），BibTeX 存 `文献记录表.md`。
