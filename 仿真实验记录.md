## 填写规则
- 每次实验一行，实验ID 唯一且递增（EXP-001、EXP-002…）。
- “关键结果”写具体数值，不要写“效果不错”这类模糊表述。
- “脚本版本”尽量记 git commit 或日期，保证能回溯到当时的代码。
- 失败 / 异常的实验也要记，注明现象和推测原因。
